# NEXARA — B2B Commerce Platform Website

NEXARA adalah landing page website modern bertema **B2B E-Commerce / Procurement Platform** yang dirancang untuk perusahaan skala besar di Indonesia.

Website ini menampilkan konsep platform pengadaan digital seperti marketplace vendor, procurement automation, business intelligence, logistics management, hingga compliance system dengan tampilan premium ala startup enterprise seperti Airbnb + SaaS enterprise platform.

## Preview

Website memiliki beberapa section utama:

- Hero Section
- Navigation Bar
- Services Section
- Procurement Process Section
- Testimonials
- Pricing Plans
- CTA Section
- Footer

## How to Run

1. Clone repository ini

```bash
git clone https://github.com/username/nexara.git
```

2. Masuk ke folder project

```bash
cd nexara
```

3. Jalankan file

```bash
open index.html
```

Atau cukup buka file `index.html` langsung di browser.
